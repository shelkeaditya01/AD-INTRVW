# AD-INTRVW - AI-Powered Mock Interview Platform

AD-INTRVW is a modern, production-ready AI-powered mock interview platform built with Next.js 15, featuring beautiful UI/UX design and full mobile responsiveness.

## ✨ Features

- 🤖 **AI-Powered Interviews** - Generate realistic interview questions using Google's Gemini AI
- 📱 **Fully Responsive** - Beautiful design that works seamlessly on all devices
- 🎨 **Modern UI/UX** - Production-level design with smooth animations and gradients
- 📊 **Performance Analytics** - Track your interview progress and get detailed feedback
- 🎥 **Webcam Integration** - Simulate real interview scenarios with video
- 🔐 **Secure Authentication** - Powered by Clerk for robust user management
- 💾 **Database Integration** - Neon PostgreSQL with Drizzle ORM

## 🚀 Tech Stack

- **Framework:** Next.js 15 with App Router
- **Styling:** Tailwind CSS 4 with custom animations
- **UI Components:** Radix UI + shadcn/ui
- **Authentication:** Clerk
- **Database:** Neon PostgreSQL + Drizzle ORM
- **AI:** Google Gemini AI
- **Icons:** Lucide React

## 📱 Mobile Responsive Design

The entire application is fully optimized for mobile devices with:
- Responsive navigation with mobile menu
- Touch-friendly buttons and interactions
- Optimized layouts for all screen sizes
- Smooth animations and transitions
